# Personal Accounting App
Система для ведення персонального фінансового обліку.

!!! Важлива інформація для перевірки/тестування !!!

На даному етапі розробки проект працює в режимі Single User (одного користувача).

Хоча функціонал реєстрації нових користувачів технічно реалізований і працює, логіка відображення рахунків та транзакцій наразі оптимізована для демонстрації на прикладі основного користувача. Створення нових акаунтів може призвести до відображення спільних даних.
